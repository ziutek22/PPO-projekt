## Folder `gui`

Folder zawiera warstwę interfejsu graficznego aplikacji (GUI), zbudowaną w oparciu o bibliotekę `tkinter`. Odpowiada za interakcję użytkownika z systemem.

### `main_window.py`
- Główne okno aplikacji.
- Umożliwia przejście do modułów: samochody, klienci, wypożyczenia.

### `cars_window.py`
- Okno zarządzania samochodami.
- Wyświetlanie listy samochodów.
- Dodawanie, edycja, usuwanie aut.
- Zmiana dostępności samochodu.

### `clients_window.py`
- Okno zarządzania klientami.
- Wyświetlanie listy klientów.
- Dodawanie i usuwanie klientów.

### `rentals_window.py`
- Okno zarządzania wypożyczeniami.
- Tworzenie nowych wypożyczeń.
- Wyświetlanie listy wypożyczeń.
- Wybór klienta i dostępnego samochodu.
